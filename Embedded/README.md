## Summary

```
"실시간 선호도 조사"은 투표를 통해 유저들끼리의 커뮤니티를 형성 할 수있는 가상의
주제로써 이 프로젝트를 통해 React, Node.js, Mysql, Hw등 다양한 기술을
접하는것을 목표로한다.
KIOSK에서 투표 기능을 개발
```
